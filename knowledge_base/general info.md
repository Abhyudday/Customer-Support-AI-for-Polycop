# PolyCop Knowledge Base (for Pinecone)

## General Rules & System Behavior

### Referral Rewards
- Referral funds/rewards are automatically transferred to the user wallet daily at **00:00 UTC** (Atleast $10)

### Wrong Chain Deposit
- If funds are deposited on the wrong chain:
  - The transaction address is a **sub-wallet generated by Polymarket**
  - This wallet is **not accessible**
  - Funds are **not recoverable**

### Bot Not Executing Trades
- Possible reasons:
  - Settings do not match current market conditions
  - Insufficient funds in wallet

### Account Import Rules
- Only **wallet-connected Polymarket accounts** can be imported
- Email-based accounts **cannot be imported**

### Redeem / Withdrawal
- Depends on Polymarket network conditions
- During congestion:
  - Redemptions may be delayed
  - Transactions will still be processed eventually

## Trading Settings

### Max Position Size
- To limit each position to $5:
  - Set **"Max per Yes/No" = $5**

### Selling Position
- Steps:
  1. Open your position

## Copy Trading Best Practices

### Important Guidelines
- Do your own research (DYOR)
- Avoid copying:
  - Wallets not suited for copy trading
  - High-frequency traders making tiny profits
  - “Perfect” equity curve wallets (likely risky)

### Key Metric
- Prefer wallets with **high Gain-to-Loss ratio**

### How to Analyze Wallet
- On desktop or landscape mode:
  - Click `!` icon → view Gain/Loss data
- Check:
  - Trading activity
  - Closed positions
  - Sort by time to analyze historical performance

## Copy Trading Strategy Principles

### Principle 1
- Match the **position size ratio** of the target wallet

### Principle 2
- Avoid using all funds too quickly
  - Prevents loss from small fluctuations

### Capital Insight
- Small capital = higher luck dependency
- Larger capital + proper strategy = more stable results
